# 常见选择语句逻辑

## 闰年判断

- 普通闰年：`y % 4 == 0 && y % 100 != 0`
- 世纪闰年：`y % 400 == 0`
- 闰年判断：`(y % 4 == 0 && y % 100 != 0) || y % 400 == 0`
  - 因为 `&&` 优先级高于 `||`，所以不写那个小括号也可以。

## 三角形判断

任意两边之和大于第三边。

`a + b > c && a + c > b && b + c > a`

## 上下取整

`int p, q, x;`，这里我们只讨论 $p,q\gt 0$ 的情况。

- 下取整（$x = \lfloor \frac{p}{q}\rfloor$）
  - `x = p / q;`
- 上取整（$\lceil \frac{p}{q}\rceil$）
  - ```cpp
    if (p % q == 0) 
        x = p / q; 
    else 
        x = p / q + 1;
    ```
  - `x = (p + (q - 1)) / q;`
  - `x = p / q + (p % q != 0);`
