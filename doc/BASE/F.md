# 整数编码

## 基础二进制

- 二进制即要求满二进一，因此每一位为 $0$ 或 $1$
- 二进制从最低位到最高位，数位的权值分别为：$2^0,2^1,2^2,2^3,\dots$
- 即 $1,2,4,8,16,32,64,128,\dots$

$(101101)_2$ 转换成十进制结果如下

```
原始数位： 1  0  1  1  0  1
对应权值：32 16  8  4  2  1
```

$32+8+4+1=45$

## 原码、反码、补码

- 原码
  - 最高位为符号位，正数为 $0$，负数为 $1$。
  - 剩下位置为原数绝对值的二进制。
- 反码
  - 正数反码与原码一致。
  - 负数反码除最高位之外，在原码基础上取反。
- 补码
  - 正数补码与原码一致
  - 负数补码在反码基础上加 $1$。
